# Projeto de Backoffice - Desafio FIAP 2024

Este projeto foi desenvolvido como parte do **challenge** da FIAP no segundo semestre do curso de **Análise e Desenvolvimento de Sistemas**. O desafio consistia em criar o **front-end** de uma aplicação web do **backoffice**, com foco em melhorar o gerenciamento e a experiência de entregadores e restaurantes no contexto de plataformas de delivery como o iFood.

## Descrição do Projeto

Nosso grupo desenvolveu uma solução para aprimorar a gestão do iFood, propondo três novas funcionalidades para o backoffice:

1. **Página de Controle de Entregadores**: Um painel de controle detalhado, onde administradores podem monitorar e gerenciar entregadores, acompanhar o status de entregas, e verificar o desempenho dos colaboradores.
2. **Página de Cardápio Detalhado com Lucros e Margens**: Uma página que oferece uma visão detalhada dos itens do cardápio, exibindo informações como lucro por item e margem de contribuição, auxiliando os restaurantes a tomar decisões estratégicas para otimizar seus lucros.

3. **Página de Gerenciamento de Empréstimos**: Uma seção dedicada ao controle de empréstimos e financiamentos concedidos aos restaurantes, permitindo um acompanhamento preciso dos valores, prazos e juros.

## Tecnologias Utilizadas

O desenvolvimento do front-end foi feito utilizando as tecnologias apresentadas no curso:

- **HTML5**
- **CSS3**
- **Bootstrap** (para design responsivo)
- **JavaScript** (Frameworks recomendados)

Todo o projeto foi criado com foco na **experiência do usuário** e em um **design responsivo**, garantindo que o sistema seja acessível tanto em desktop quanto em dispositivos móveis.
